What the product actually does

OwlyVision is a brief system, not a generic website audit.

Generate a pitch brief

Paste in a domain, capture live signals, and turn the result into a structured brief instead of a raw scrape.

Track high-value targets

Move important companies onto the watchlist so you can refresh the brief and keep prospect context current.

Deliver the brief cleanly

Share it privately, export it as PDF, and keep basic pipeline and outreach state close to the brief itself.

Current beta

The docs describe the live split-domain beta.

Marketing is intended for owlyvision.com, the app is live on app.owlyvision.com, and docs are intended for docs.owlyvision.com. Internal runbooks can stay in-repo; this docs surface is meant to stay public and product-facing on a dedicated static host.
